Adds checkboxes to each widget to either show or hide on every site page.

== Description ==

Avoid creating multiple sidebars and duplicating widgets by adding check boxes to each widget which will either show or hide the widgets on every site page. Great for use with Thesis theme, or just to avoid extra coding.

== Installation ==

1. Upload `display-widgets.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
